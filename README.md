# pdffonts

A command-line tool to display fonts embedded inside PDF documents.

## Installation

    $ gem install pdffonts

## Usage

    $ pdffonts pdf-with-beautiful-type.pdf
    Avenir-Black
    Avenir-Medium
    Avenir-Roman
    AvenirNext-DemiBold
    GearedSlab-Extrabold
    Lavanderia-Sturdy
    LucidaGrande
    LucidaGrande-Bold
    Mission-Script
    ZapfDingbatsITC
